# Alna Elsa Tom - Portfolio

Welcome to my personal portfolio website! This site showcases my skills, projects, hackathon experiences, and contact information. Built with HTML, CSS, and JavaScript, it features a responsive design and interactive elements.

## Features

- **About Me:** Learn about my background, interests, and education.
- **Tech Stack:** Visual display of technologies and tools I use.
- **Projects:** Highlights of my key projects with links to GitHub repositories.
- **Chronicles:** Hackathon and event experiences, presented in a Swiper carousel.
- **Contact:** Easy way to reach out via email or download my CV.
